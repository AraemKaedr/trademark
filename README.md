# trademark
Тема ВКР: Разработка программы поиска графических товарных знаков с использованием искусственной нейронной сети

---

## Установка и запуск

### 1. Клонирование репозитория

Перейдите в папку, в которой собираетесь работать с проектом. 

В терминале введите команды:

```git clone https://github.com/AraemKaedr/trademark.git```

```cd trademark```

### 2. Создание виртуального окружения

Для создания виртуального окружения напишите в терминале:

1. для Linux:

```python3 -m venv venv```

2. для Windows

```python -m venv venv```

Затем, чтобы активировать виртуальное окружение пропишите код

1. для Linux:

    - ```source venv/bin/activate```

2. для Windows

    - ```venv\Scripts\activate```

    или

    - ```venv\Scripts\activate.bat```

### 3. Установка зависимостей

Все используемые библиотеки находятся в файле requirements.txt.

Для их установки вам нужно прописать команду

```pip install -r requirements.txt```

### 4. Подготовка данных и моделей

При первом запуске ```main.py``` автоматически предложит скачать датасет и модели.

Также можно запустить подготовку вручную:

**Скачивание и распаковка датасета**

```python download_dataset.py```

**Скачивание моделей SAM и YOLO**

```python download_models.py```

### 5. Запуск приложения

Для запуска приложения введите в терминал

```python main.py```

---

## Основные возможности

- Очистка и предобработка датасета (поддержка .png, .jpg, .jpeg, .webp, .gif)
- Извлечение эмбеддингов с помощью **SAM ViT-B** + **YOLOv8n**
- Построение векторного индекса **FAISS**
Кластеризация логотипов
- Поиск похожих товарных знаков с указанием процента сходства
- Удобный графический интерфейс (PyQt6)
